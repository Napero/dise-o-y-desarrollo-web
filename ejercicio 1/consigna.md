Utilizando las etiquetas HTML vistas en clase, realiza los siguientes ejercicios: 
 
1. Tu primera página web: 
Objetivo: Familiarizarte con la estructura básica de un documento HTML. 
Qué hacer: En un documento HTML llamado 01.html, escribir una página web simple con un título, un párrafo de texto, como por ejemplo una presentación breve tuya. 
Mínimo de etiquetas a usar: <!DOCTYPE html>, <html>, <head>, <title>, <body> y etiquetas de contenidos. 
 
2. Formateando tu texto: 
Objetivo: Practicar con etiquetas de encabezado, negrita, cursiva y saltos de línea. 
Qué hacer: En un documento HTML llamado 02.html, escribir un texto con diferentes formatos, incluyendo títulos, párrafos, palabras en negrita y cursiva, y saltos de línea. Puede ser la transcripción de una noticia corta de actualidad. 
Mínimo de etiqueta a usar: <h1>, <h2>, <strong>, <em>, <br> y <p>. 
 
3. Listas de elementos: 
Objetivo: Aprender a usar listas ordenadas y desordenadas. 
Qué hacer: En un documento HTML llamado 03.html, crear una lista de elementos, como una lista de ingredientes para una receta y luego una lista de pasos para ejecutarla. 
Etiqueta a usar: <ol>, <ul>, <li> y <li> para elementos de la lista. 
 
4. Insertando una imagen: 
Objetivo: Familiarizarte con la etiqueta <img> y sus atributos. 
Qué hacer: En un documento HTML llamado 04.html, insertar una imagen en tu página web, como una foto de tu mascota o un paisaje. Para esto, la imagen deberá estar guardada dentro de una subcarpeta llamada imgs en tu proyecto. 
Etiqueta a usar: <img> con los atributos src, alt y width. 
 
5. Enlaces a otras páginas web: 
Objetivo: Practicar con la creación de hipervínculos. 
Qué hacer: Agregar un enlace externo en cada uno de tus documentos anteriores que te lleve a otra página web, como tu página web personal o un sitio web de noticias, una tienda online donde comprar los ingredientes de tu receta, tus redes sociales favoritas, etcétera. También, en cada documento crea una lista desordenada con los enlaces a cada uno de los ejercicios realizados. Ubica esta lista la final del documento. 
Etiqueta a usar: <a> con el atributo href y/o target.
